# Form Projek Akhir

## Judul Projek Akhir

SIRS - Sistem Informasi Respons Insiden Keamanan Siber

## Deskripsi Judul/Aplikasi

SIRS adalah aplikasi Android untuk pelaporan dan penanganan insiden keamanan siber. Pengguna dapat membuat laporan insiden dengan menyertakan kategori, tingkat keparahan, deskripsi kejadian, lokasi, tanggal kejadian, dan bukti pendukung.
Admin dapat memantau laporan yang masuk, mengubah status dan tingkat keparahan laporan, memberikan respon penanganan, mengelola kategori, serta melihat riwayat aktivitas aplikasi. Aplikasi ini membantu proses pelaporan insiden menjadi lebih terstruktur, mudah dipantau, dan terdokumentasi dengan baik.

## Fitur Aplikasi

1. Login dan registrasi akun
2. Reset password melalui email
3. Dashboard pengguna
4. Dashboard admin
5. Pembuatan laporan insiden keamanan siber
6. Pengisian data laporan seperti judul, kategori, tingkat keparahan, deskripsi, lokasi, dan tanggal kejadian
7. Upload bukti pendukung laporan
8. Daftar laporan milik pengguna
9. Detail laporan pengguna
10. Filter laporan berdasarkan status
11. Filter laporan berdasarkan tingkat keparahan
12. Edit laporan oleh pengguna
13. Hapus laporan oleh pengguna
14. Daftar seluruh laporan untuk admin
15. Pencarian laporan oleh admin
16. Filter laporan admin berdasarkan status, tingkat keparahan, dan tanggal
17. Detail laporan admin
18. Perubahan status laporan oleh admin
19. Perubahan tingkat keparahan laporan oleh admin
20. Respon atau catatan penanganan dari admin
21. Hapus laporan oleh admin
22. Riwayat perubahan status laporan
23. Manajemen kategori laporan
24. Pencatatan aktivitas penting pada aplikasi
25. Perhitungan batas waktu penanganan berdasarkan tingkat keparahan laporan

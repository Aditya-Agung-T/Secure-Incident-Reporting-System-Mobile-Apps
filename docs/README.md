# Dokumentasi Project SIRS

Folder ini berisi dokumentasi pendukung untuk penilaian project akhir.

## Daftar Dokumen

1. `PROJECT_OVERVIEW.md`  
   Ringkasan aplikasi, tujuan, fitur, dan struktur umum project.

2. `SECURITY_FEATURES.md`  
   Ringkasan fitur keamanan yang diterapkan pada aplikasi.

3. `SETUP_AND_RUN.md`  
   Panduan menjalankan project di Android Studio.

4. `form-project-akhir.md`  
   Draft isian form project akhir.

## Catatan

Dokumentasi ini ditulis untuk kebutuhan penilaian UAS praktikum. Penjelasan dibuat ringkas dan fokus pada fungsi aplikasi, cara menjalankan, serta fitur keamanan yang terlihat dan relevan.
